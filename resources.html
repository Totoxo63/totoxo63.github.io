🛠️ Librairies Python

👶 Librairies standards

L’ensemble des librairies standards de Python est consultable ici : https://docs.python.org/3/library/index.html. Celles principalement utilisées sont les suivantes :

math : https://docs.python.org/3/library/math.html Fonctions mathématiques. Voir cmath pour des fonctions pour les nombres complexes.

random : https://docs.python.org/fr/3/library/random.html Génération de nombres aléatoires. Fonctionnalité aussi disponible dans d’autres librairies scientifiques.

os et os.path : https://docs.python.org/fr/3.7/library/os.html Interface avec le système d’exploitation. Très utilisé pour la gestion des fichiers et des chemins de fichier.

Pathlib : https://docs.python.org/3/library/pathlib.html Librairie pour gérer les chemins d’accès de fichier sans se préoccuper des spécificités liées au système d’exploitation utilisé.

argparse : https://docs.python.org/3/library/argparse.html Module de création d’interface en ligne de commande (CLI).

time : https://docs.python.org/3/library/time.html Mesure du temps d’exécution d’une portion de code.

subprocess : https://docs.python.org/3/library/subprocess.html Exécution de lignes de commande depuis Python.

🔬 Librairies scientifiques usuelles

NumPy : https://numpy.org/ Librairie fondamentale pour le calcul sur des tableaux multidimensionnels.

Matplotlib : https://matplotlib.org/ Création de figures et graphiques en Python.

SciPy : https://scipy.org/ Librairie de calcul mathématique : optimisation, intégration, interpolation, équations algébriques, etc.

OpenCV : https://opencv.org/ Librairie pour le traitement d’image et la vision par ordinateur.

🧠 Librairies de Machine Learning (ML) et Deep Learning (DL)

scikit-learn : https://scikit-learn.org/ Librairie de ML pour les taches de classification, régression, clustering, réduction de dimensionalité, etc.

PyTorch : https://pytorch.org/ Librairie de DL créée par Meta, très utilisée en recherche.

TensorFlow : https://www.tensorflow.org/ Librairie de DL créée par Google. Souvent utilisé avec Keras.

Keras : https://keras.io/ Framework Python de DL qui permet une manipulation de haut niveau, initialement pour le backend TensorFlow, mais supporte également PyTorch et JAX depuis Keras 3.

JAX : https://jax.readthedocs.io/ Librairie de DL orientée pour le calcul accéléré (JIT, XLA).

PyTorch Lightning : https://lightning.ai/docs/pytorch/stable/ Framework pour PyTorch pour simplifier l’écriture de code.

🎲 Autres librairies utiles

TQDM : https://tqdm.github.io/ Création de barres de chargement.

Soundfile : https://python-soundfile.readthedocs.io/ Lecture et écriture de fichiers audio.

Einops : https://einops.rocks/ Librairie pour faciliter et rendre plus lisible la manipulation des tenseurs, en particulier pour les transpositions.

TorchInfo : https://github.com/tyleryep/torchinfo Librairie permettant d’afficher l’architecture d’un modèle et la taille des tenseurs à chaque couche du réseau quand on utilise PyTorch.

Broadcasting : https://numpy.org/doc/stable/user/basics.broadcasting.html Utilisation puissante des tenseurs NumPy pour certains calculs (aussi valable pour TensorFlow et PyTorch).
